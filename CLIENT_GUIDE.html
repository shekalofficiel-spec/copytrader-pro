Deuxième étape

Ajouter vos comptes MT5

Depuis votre tableau de bord, vous allez enregistrer chacun de vos comptes MT5. Un compte sera désigné MASTER (le compte où vous tradez), les autres seront désignés SLAVE (les comptes qui copient).

  1. 1
    Aller dans "Accounts" Dans le menu latéral du dashboard, cliquez sur Accounts.
  2. 2
    Cliquer sur "Add Account" Le formulaire d'ajout de compte s'ouvre.
  3. 3
    Remplir les informations pour chaque compte Voir les champs détaillés ci-dessous.
  4. 4
    Répéter pour chaque compte MT5 Ajoutez d'abord le MASTER, puis chaque SLAVE un par un.

Détail des champs à remplir

Nom du compte
Un nom lisible, ex: "Mon Exness Master" ou "FTMO Compte 1"
Broker
Saisissez METAAPI — ce champ est utilisé pour l'identification interne
Rôle
MASTER — votre compte principal (vous tradez dessus)
SLAVE — les comptes qui copient automatiquement
Ratio
1.0 = copie exacte des lots  |  0.5 = moitié des lots  |  2.0 = double des lots
Important — Rôles
⚠ Vous ne pouvez avoir qu'un seul compte MASTER actif à la fois. Tous vos autres comptes doivent être en mode SLAVE.

Troisième étape

Générer les tokens EA

Chaque compte nécessite un token d'authentification unique. Ce token est utilisé par l'EA dans MT5 pour communiquer avec le serveur CopyTrader Pro de façon sécurisée.

  1. 1
    Sélectionner un compte dans la liste Cliquez sur le compte pour ouvrir ses détails.
  2. 2
    Cliquer sur "Generate EA Token" Un token JWT est généré et affiché à l'écran.
  3. 3
    Copier et conserver le token Copiez-le immédiatement dans un endroit sécurisé (ex: un document local). Il ne sera affiché qu'une seule fois en clair.
  4. 4
    Copier l'Account ID L'Account ID est visible sur la même page — notez-le également.
  5. 5
    Répéter pour chaque compte Chaque compte (MASTER et chaque SLAVE) a son propre token et Account ID.

Sécurité des tokens
⚠ Traitez vos tokens comme des mots de passe. Ne les partagez jamais par email ou messagerie non sécurisée.
⚠ Si un token est compromis, revenez sur le dashboard et régénérez-en un nouveau.

Quatrième étape

Installer l'EA dans MetaTrader 5

L'Expert Advisor (EA) est le programme qui s'exécute directement dans MT5 et qui assure la communication avec le serveur CopyTrader Pro. Cette installation se fait une seule fois sur chaque ordinateur.

  1. 1
    Télécharger le fichier EA Récupérez le fichier CopyTradeBridgeMT5.mq5 auprès de votre gestionnaire.
  2. 2
    Ouvrir le dossier des données MT5 Dans MetaTrader 5 : Fichier → Ouvrir le dossier des données
    Ou appuyez sur Ctrl+Shift+D
  3. 3
    Naviguer vers le dossier Experts Dans le dossier qui s'ouvre : MQL5 → Experts
  4. 4
    Coller le fichier .mq5 Copiez CopyTradeBridgeMT5.mq5 dans ce dossier Experts.
  5. 5
    Autoriser les requêtes WebRequest Dans MT5 : Outils → Options → onglet Expert Advisors
    Cocher "Autoriser les requêtes WebRequest pour les URLs listées" puis ajouter l'URL ci-dessous.
URL à autoriser dans MT5
https://copytrader-pro-production.up.railway.app
  1. 6
    Compiler l'EA De retour dans MT5, dans le Navigateur : faites un clic droit sur CopyTradeBridgeMT5 → Modifier. L'éditeur MetaEditor s'ouvre. Appuyez sur F7 pour compiler.
    La compilation doit se terminer sans erreur (0 erreur, 0 warning).
Compilation réussie
✓ La barre en bas de MetaEditor affiche 0 erreur(s), 0 avertissement(s). L'EA est prêt à être utilisé.

Cinquième étape

Configurer l'EA sur chaque compte

Cette étape doit être réalisée séparément sur chaque compte MT5 (MASTER et chaque SLAVE). Connectez-vous au bon compte MT5 avant de commencer.

Sur le compte MASTER

  1. 1
    Ouvrir un graphique Double-cliquez sur n'importe quelle paire (ex: EURUSD H1) pour ouvrir un graphique.
  2. 2
    Glisser l'EA sur le graphique Dans le Navigateur (panneau gauche), sous Expert Advisors, trouvez CopyTradeBridgeMT5 et faites-le glisser sur le graphique ouvert.
  3. 3
    Remplir les paramètres Une fenêtre de paramètres s'ouvre — configurez-la comme suit :
BackendURL
https://copytrader-pro-production.up.railway.app
AccountToken
Votre token MASTER (commençant par eyJ...)
AccountID
Votre Account ID MASTER (ex: abc-1234-...)
Mode
MASTER
  1. 4
    Cliquer OK L'EA se lance. Dans le coin supérieur droit du graphique, vous devez voir :
    ✅ CopyTrader Pro MASTER actif

Sur chaque compte SLAVE

La procédure est identique, mais avec les informations propres au compte SLAVE concerné.

BackendURL
https://copytrader-pro-production.up.railway.app
AccountToken
Le token de ce compte SLAVE
AccountID
L'Account ID de ce compte SLAVE
Mode
SLAVE
✓ Message attendu : ✅ CopyTrader Pro SLAVE actif
⚠ Chaque compte SLAVE a ses propres token et Account ID. Ne mélangez pas les tokens entre comptes.

Sixième étape

Tester la connexion

Avant de commencer à trader normalement, effectuez un test rapide pour valider que la copie fonctionne correctement.

  1. 1
    Ouvrir un micro trade sur le compte MASTER Sur le compte MASTER (avec l'EA actif), ouvrez manuellement un trade de 0.01 lot sur n'importe quelle paire. Utilisez un trade de test — la taille minimale suffit.
  2. 2
    Vérifier les comptes SLAVE dans les 2 secondes Basculez sur vos comptes SLAVE. Le même trade doit apparaître automatiquement (avec les lots ajustés selon le ratio configuré).
  3. 3
    Vérifier le dashboard CopyTrader Pro Ouvrez le dashboard dans votre navigateur — le trade en cours doit être visible en temps réel.
https://frontend-pearl-nu-68.vercel.app
  1. 4
    Clôturer le trade de test Clôturez le trade sur le compte MASTER. Il doit se clôturer automatiquement sur tous les comptes SLAVE.
Test réussi — Système opérationnel
✓ Le trade est apparu sur tous les SLAVE en moins de 2 secondes.
✓ La clôture sur le MASTER a entraîné la clôture sur tous les SLAVE.
✓ Le dashboard affiche l'activité en temps réel.

Avertissements et bonnes pratiques

⚠ Fonctionnement de l'EA — Conditions requises
⚠ L'EA doit rester actif en permanence — MetaTrader 5 doit rester ouvert sur tous les comptes (MASTER et SLAVE) pour que la copie fonctionne. Si MT5 est fermé ou l'EA retiré, les trades ne seront plus copiés.
⚠ Pensez à utiliser un VPS si vous avez besoin d'un fonctionnement 24h/24. Demandez conseil à votre gestionnaire.
⚠ Comptes SLAVE — Ne pas intervenir manuellement
⚠ Ne modifiez pas manuellement les trades sur les comptes SLAVE. Les modifications manuelles (SL, TP, clôture partielle) peuvent créer des désynchronisations et générer des erreurs dans le système de copie.
⚠ Sécurité — Tokens confidentiels
⚠ Vos tokens EA sont confidentiels. Ne les partagez pas, ne les envoyez pas par email non sécurisé, et ne les publiez jamais en ligne. En cas de doute sur la sécurité d'un token, régénérez-en un nouveau depuis le dashboard.
📧 Support — Contactez votre gestionnaire
✓ Pour toute question, problème technique ou demande de configuration, contactez directement votre gestionnaire CopyTrader Pro.
✓ Préparez les informations suivantes lors de votre prise de contact : votre email de compte, le nom des comptes concernés, et une description précise du problème rencontré.

En cas d'urgence

Kill Switch — Arrêt d'urgence

Si vous devez stopper immédiatement tous les trades sur tous les comptes, CopyTrader Pro propose un Kill Switch accessible en un clic depuis le dashboard.

⛔ Attention — Action irréversible
⛔ Le Kill Switch clôture tous les trades ouverts sur tous les comptes simultanément, aux prix du marché. Cette action ne peut pas être annulée. N'utilisez cette fonction qu'en cas d'urgence réelle.

Méthode 1 — Via le Dashboard (recommandée)

  1. 1
    Ouvrir le dashboard CopyTrader Pro https://frontend-pearl-nu-68.vercel.app
  2. 2
    Localiser le bouton rouge "Kill Switch" Il est visible en haut de votre dashboard, dans la barre de contrôle.
  3. 3
    Cliquer et confirmer Une confirmation est demandée. Confirmez pour fermer tous les trades instantanément.

    ⏹ KILL SWITCH

Méthode 2 — Via MetaTrader 5 (si le dashboard est inaccessible)

  1. 1
    Sur chaque compte MT5 Faites un clic droit sur le graphique où l'EA est actif → Expert Advisors → Retirer.
  2. 2
    Clôturer les trades manuellement Une fois l'EA retiré, clôturez manuellement les trades ouverts depuis l'onglet Trade dans MT5.
